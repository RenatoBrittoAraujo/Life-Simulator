# SDL2 Game Template

Including:
- Makefile
-	Game class
- Main
- Util class with useful functions
- Color class
- Graphics class
- Input handling and abstraction
- Sprite creating class
- Point and Segment classes

Running normally:
```
make run
```

Running verbose:
```
make runv
```

Running with set framerate:
```
make run framerate=60
```
